# Модель для предсказания массы морских черепах
Проект был выполнен в ходе обучения в Яндекс.Практикуме, профессия "Специалист по Data Science".

#### Краткое описание проекта
Необходимо создать модель для предсказания массы морских черепах. Причиной создания модели служит отсутствие возможности взвешивания тяжелых черепах в естественных условиях.

#### Реализованные этапы проекта
- Исследовательский анализ данных;
- Предобработка данных;
- Обучены модели `LinearRegression`, `Lasso`, `Ridge`, `SGDRegressor`;
- Проведено сравнение результатов;
- Оценены вклады признаков;
- Сделаны выводы на основе работы.

#### Основной использованный в проекте стек технологий:
- **Язык**: Python
- **Обработка данных**: pandas, NumPy
- **Машинное обучение**: scikit-learn
- **Визуализация**: matplotlib, seaborn
- **Модели**: LinearRegression, Ridge, Lasso, SGDRegressor (DummyRegressor в качестве базовой модели)
